Ця проблема виникає через те, що при деплої на GitHub Pages сервер не знає, як обробляти маршрутизацію Vue, оскільки це односторінковий додаток (SPA). Коли ви переходите на https://acvetochka.github.io/weather-app/about, сервер намагається знайти фізичний файл /about, якого не існує, і тому повертає помилку 404. Щоб виправити це, потрібно налаштувати 404.html файл так, щоб він перенаправляв всі запити на ваш index.html файл, який потім оброблятиметься маршрутизатором Vue. GitHub Pages підтримує цю техніку.
